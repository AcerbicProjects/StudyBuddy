# 📚 StudyBuddy

*Transforming passive reading into active recall through intelligent, automated study tools.*

**StudyBuddy** is an AI-powered mobile application designed to help university students intelligently manage, understand, and master their academic coursework. Built as part of the **CSE 4675: Mobile App Development** course, this platform tackles academic burnout by bridging the gap between complex lecture slides and genuine conceptual mastery. The app leverages multimodal AI to instantly convert dense lecture PDFs and handwritten notes into interactive study materials, tailored viva questions, and personalized revision schedules.

## ✨ Key Features

* **🤖 AI Study Companion Chatbot:** A 24/7 contextual teaching assistant that adapts explanation styles to the user's skill level and answers complex course-related questions instantly.
* **📄 Smart OCR Document Hub:** Upload lecture slides, PDFs, or snap photos of handwritten notes/math problems. The app extracts the text, generates quick summaries, and formats structured flashcards.
* **🧠 Dynamic Assessment Engine:** Automatically generates high-yield mock evaluations, including hard-level quizzes and viva questions directly from uploaded documents.
* **📊 Weak-Topic Analytics:** An intelligent workspace and dashboard that tracks learning behavior, past quiz mistakes, and overall subject progress.
* **⏰ Smart Recommendations:** Pushes targeted reminders and personalized daily revision schedules based on topic difficulty and upcoming exams.

## 🛠️ Tech Stack

**Frontend**
* **React Native:** For a fast, responsive, cross-platform (iOS/Android) mobile interface featuring a distraction-free dark/light mode layout.

**Backend & Database**
* **Node.js:** Backend environment handling microservices and AI API endpoint orchestration.
* **Supabase / Firebase:** Secure student authentication, real-time document reference syncing, and database management.

**AI Ecosystem**
* **Gemini API / OpenAI API:** Core Large Language Models (LLMs) for natural language processing, semantic understanding, and text generation.
* **Google Vision API / OCR.space:** Multimodal pipeline for high-accuracy Optical Character Recognition (OCR) to process images and PDFs.

## 🚀 Example Use Cases

* **Snap & Explain:** Snap a photo of a tough math formula, and the AI explains it in simple language.
* **PDF to Viva Prep:** Upload a 40-page lecture PDF and instantly receive 15 high-yielding viva questions.
* **Targeted Revision:** Track past quiz mistakes and receive smart revision reminders for weak topics.
